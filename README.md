# Подготовка к собеседованиям по Go

Личный репозиторий для подготовки к собесам на Go-разработчика.

## Что внутри

- `theory/` — вопросы и ответы по темам
- `go-interview-tasks/` — практические задачи
- `docs/` — PDF со всей теорией
- `cv/` — резюме

Список всех вопросов — в `theory/questions.md`.

## Как пользоваться

Читать теорию в `theory/` или PDF из `docs/`. Задачи решать в `go-interview-tasks/`.

Проверить решения:

```bash
./scripts/check.sh
```

Собрать PDF локально:

```bash
./scripts/build_docs.sh
```
